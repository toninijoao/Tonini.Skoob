🦉 Tonini Skoob
Inspirado na rede social de leitores Skoob, o Tonini Skoob é um organizador de leituras automatizado. Ele permite cadastrar livros informando apenas o título, enquanto uma Inteligência Artificial preenche os detalhes técnicos para você.

-Funcionalidades

* Cadastro Automatizado: Usa a API da Groq (Llama 3) para gerar resumos e dados técnicos.
* Banco de Dados Local: Armazena suas leituras usando SQLite.
* Estatísticas de Leitura: Calcula automaticamente o total de livros lidos e a soma de páginas.
* Relatório em TXT: Gera um arquivo txt formatado para visualização da sua estante.
* Gestão Completa: Permite adicionar, remover e listar livros através de um menu interativo.

-Tecnologias Utilizadas

* Python
* SQLite3: Banco de Dados
* Groq Cloud API (LLM)
